PEW RESEARCH CENTER
Wave 49 American Trends Panel 
Dates: June 3-June 17, 2019
Mode: Web 

Sample: Subsample
Language: English and Spanish
N=4,272


***************************************************************************************************************************
NOTES

The following coded variables are included in the W49 dataset:

SNSUSER_W49 - Social media use status (yes/no)
TRACKCNT_W49 - Number of devices/services used that utilize personal data (total of five, including smart speaker, other connected home devices, fitness tracker, grocery loyalty programs, and mail-in DNA test)
TOTALKNOW_W49 - Total number of knowledge questions answered correctly (maximum of 10)


***************************************************************************************************************************
WEIGHTS 

WEIGHT_W49 is the weight for the sample. Data for all Pew Research Center reports are analyzed using this weight.
